A Java-based "Student Management System" designed to manage student records efficiently.
This project allows users to 'add, view, update, delete, and search student details' using a simple and structured approach.
It is developed as a "mini project" to understand core Java concepts and basic database/file handling.

Features

* Add new student records
* View all student details
* Update existing student information
* Delete student records
* Search students by ID or name
* User-friendly and menu-driven interface

---

##  Technologies Used

* **Programming Language:** Java
* **IDE:** IntelliJ IDEA / Eclipse / NetBeans
* **Concepts Used:**

  * OOP (Classes, Objects, Encapsulation)
  * Collections / File Handling / JDBC *(based on your version)*
  * Exception Handling
  * Basic Input/Output

##  How It Works

1. User selects an option from the menu
2. System performs the selected operation
3. Student data is stored and managed efficiently
4. Changes are reflected immediately

---

##  How to Run the Project

1. Clone or download the project
2. Open it in your preferred Java IDE
3. Compile and run the `Main.java` file
4. Follow on-screen instructions

##  Sample Operations

* **Add Student:** Enter ID, Name, Age, Course
* **View Students:** Displays all stored records
* **Update Student:** Modify existing details
* **Delete Student:** Remove a student record
* **Search Student:** Find student by ID or name
